# Staff Bus Management System

This project was developed as a **2nd Year Software Engineering Group Project** at Kotelawala Defence University – Southern Campus.

The system aims to digitize the existing manual staff bus management process and improve the commuting experience for students and lecturers.

## Features

- Online Seat Reservation
- Real-Time GPS Bus Tracking
- Automated Notifications
- Lost & Found Management
- Feedback and Complaint System
- Ticket Rollover System
- Admin Dashboard for Managing Bus Details

## Technologies Used

- React Native (Mobile Application)
- Firebase (Database & Backend)
- Google Maps API (GPS Tracking)
- JavaScript
